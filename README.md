## 💡 Возможности

- 💎 Просмотр коллекции минералов (10+ видов)
- 🖼 Загрузка изображений (с fallback-заглушкой)
- 📊 Характеристики каждого минерала:
  - Цвет  
  - Структура  
  - Твердость  
  - Ценность  
- ⭐ Система оценки (звёзды)
- 👁 Счётчик просмотров для каждого минерала
- 🎲 Случайные интересные факты
- 🌗 Современный тёмный интерфейс
- 🧭 Навигация через вкладки (Tkinter Notebook)
