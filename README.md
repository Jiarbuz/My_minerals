# 💎 Коллекция минералов 

[![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)](https://www.python.org/)
[![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)](https://docs.python.org/3/library/tkinter.html)
[![Pillow](https://img.shields.io/badge/Image-Pillow-orange)](https://pillow.readthedocs.io/)

> 🧪 Десктопное приложение на Python для просмотра коллекции минералов с красивым интерфейсом, изображениями и характеристиками.

## 💡 Возможности

* 💎 Просмотр коллекции минералов (10+ видов)
* 🖼 Загрузка изображений (с fallback-заглушкой)
* 📊 Характеристики каждого минерала:

  * Цвет
  * Структура
  * Твердость
  * Ценность
* ⭐ Система оценки (звёзды)
* 👁 Счётчик просмотров для каждого минерала
* 🎲 Случайные интересные факты
* 🌗 Современный тёмный интерфейс
* 🧭 Навигация через вкладки (Notebook)

## 🚀 Как запустить

1. Установи зависимости:

```bash
pip install pillow
```

2. Запусти проект:

```bash
python main.py
```

---

## ⚠️ Важно

* Папка `images/` создаётся автоматически, если её нет
* Если изображение отсутствует — появится заглушка
* Работает на Windows / Linux / macOS
